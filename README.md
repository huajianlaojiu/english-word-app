# 📚 小学生英语单词学习应用

> **让背单词变得有趣 · An English word learning app for kids**

[📦 GitHub](https://github.com/huajianlaojiu/english-word-app)

---

## 📖 简介

专为小学生设计的英语单词学习应用。通过卡片式学习和趣味测试，帮助孩子轻松掌握英语单词。

## ✨ 功能特性

- 📖 **单词卡片**：图文并茂的单词学习卡片
- 🔊 **发音功能**：点击播放标准英语发音
- 🎮 **趣味测试**：选择题、拼写题等多种测试形式
- 📊 **学习统计**：记录学习进度和正确率
